# 祖师残卷 · Sect of Broken Legacy

暗黑武侠仙侠 × 宗派养成 × 剧情驱动 H5 小游戏。
七十年内，把这门派从七小派之末抬到三大宗之一——或成尘埃。

## Play

直接打开 `index.html`，或访问 GitHub Pages 链接。

## 玩法

- 6 位起始弟子（含上代祖师 + 神秘客卿）
- 资质五维 / 境界八阶 / 寿命系统 / 渡劫小游戏 / 师兄妹羁绊
- 12+ 派遣任务（采药 / 灭妖 / 护镖 / 谈判 / 探秘 / 宗门战）
- 7 节主线剧情，前 3 日依次：地窖之兆 → 大弟子之死 → 雷霆挑衅
- 5 处宗门建筑：祖师堂 / 丹房 / 演武场 / 藏经阁 / 灵田
- 闭关修炼 / 突破 / 渡劫（3 轮心魔小游戏）
- 限定弟子 / 宗门皮肤 / 月卡 / 季票

## Stack

纯 HTML / CSS / JavaScript。立绘和场景使用 Pollinations API 生成的厚涂油画风。
WebAudio 程序化音效。LocalStorage 存档。无依赖。
